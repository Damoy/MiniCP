# MiniCP
Mini constraint programming solver

# Origin
This solver was forked. Please refer to https://www.info.ucl.ac.be/~pschaus/minicp .

# Changes
Worked on:

- Constraints

  - Element1D
  - IsLessOrEqual
  - Not Equal
  
- Examples
  - TSP
  - TSPMinRegret
  - TSPMinRegretLNS
  
- Search
  - DFSearch
  - Selector
